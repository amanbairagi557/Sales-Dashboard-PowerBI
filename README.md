Power BI Dashboard Project

Overview

This project demonstrates an interactive Power BI dashboard created for data analysis and business insights.

The dashboard enables users to analyze key performance metrics using dynamic visualizations and filters.

Objectives

- Analyze business performance
- Identify trends and patterns
- Monitor KPIs
- Support data-driven decision making


Tools Used

- Power BI Desktop
- Microsoft Excel / CSV
- Power Query
- DAX


Dashboard Features

- KPI Cards
- Bar Charts
- Line Charts
- Pie Charts
- Slicers
- Interactive Filters
- Trend Analysis


 Key Insights

- Top-performing categories identified
- Monthly sales trend analysis
- Regional performance comparison
- Customer behavior insights
- Overall business performance summary
